# PSSalon

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter application.

PS Salon is a cross‑platform mobile application available for both iOS and Android, designed to make salon appointment scheduling simple, fast, and accessible. Users can browse services, explore multiple store locations, and book appointments with ease—all from their mobile device.

✨ Key Features
Multi‑Location Support — Users can view and select from different salon branches.

Easy Appointment Booking — Choose services, pick a time slot, and confirm appointments in just a few taps.

Cross‑Platform Availability — Fully functional on both iOS and Android devices.

User‑Friendly Interface — Clean, intuitive design focused on smooth navigation and quick actions.
